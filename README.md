# This repository contains Python programs based on Decision Structures and Boolean Logic. The files demonstrate how programs make decisions using conditions, comparisons, and logical expressions.

The focus is on building problem-solving skills using if statements and understanding how Python evaluates true/false conditions.

The programs in this repository are based on the following concepts:
- Boolean Expressions and Relational Operators
- The `if` Statement
- The `if-else` Statement
- Comparing Strings
- Nested Decision Structures
- The `if-elif-else` Statement
- Logical Operators (`and`, `or`, `not`)
- Boolean Variables

These exercises help develop the following skills:
- Writing conditions using relational operators
- Making decisions in programs using if statements
- Handling multiple conditions using elif chains
- Combining conditions using logical operators
- Comparing string values correctly
- Structuring nested decision logic
- Understanding true/false (Boolean) logic in programming

Types of Programs Included
- Number classification programs (positive/negative/zero)
- Grade and marks decision programs
- Temperature or category classifiers
- String comparison programs
- Simple rule-based decision systems
- Logical condition-based problem solvers
